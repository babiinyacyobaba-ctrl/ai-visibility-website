課題

生成AIは今、
誰かに代わって
企業を推薦している

「東京でおすすめのホテルは？」「予算10万円の高級旅館は？」—ユーザーは今、検索ではなくAIに聞くようになっています。

そのとき、AIが参照するのは企業サイトに埋め込まれた構造化データです。データが整備されていない企業は、推薦リストから外れてしまいます。

これは単なる技術的な問題ではありません。集客の機会損失です。

AIの回答（構造化データあり）
帝国ホテル東京は1890年創業の老舗高級ホテルです。客室数は931室、東京都千代田区に位置し、チェックインは15:00からです。レストランは17店舗を有し、フレンチ・日本料理など多彩です。
AIの回答（構造化データなし）
〇〇旅館については詳細な情報が確認できませんでした。公式サイトをご確認ください。

評価基準

5軸・100点満点の
スコアリング

0 — 30点

基本メタ

noindex・meta description・canonical・OGPタグなど、検索エンジンとAIへの基本情報を評価。欠けているとAIはそのページを「存在しないもの」として扱います。

0 — 20点

JSON-LD保有

企業名・住所・営業時間などをAIが読める形式（JSON-LD）で記述しているかを評価。あるとAIが企業情報を正確に把握し回答に反映できます。

0 — 30点

スキーマ型

JSON-LDの中でHotel・Restaurant・Productなど業界に適したスキーマ型が設定されているかを評価。種類の網羅性と業界適合性を採点します。

0 — 20点

フィールド充実度

スキーマの中に実際の値（住所・電話番号・料金・画像URLなど）がどれだけ埋まっているかを評価。定義だけあって中身が空では意味がありません。

よくある質問

FAQ

スコアはどのように計算されますか？
各企業の公式Webサイトを自動取得し、meta情報・JSON-LD構造化データ・スキーマ型・フィールド充実度の4軸を解析して100点満点で採点します。AIが企業情報を読み取れるかどうかを客観的に数値化した指標です。
診断に使用するデータはリアルタイムですか？
はい。スキャン実行時点での各企業Webサイトのデータをリアルタイムで取得・解析します。サイトが更新されれば次回のスキャンに反映されます。
どの業界に対応していますか？
現在、ホテル・旅館、レストラン・飲食、ECサイト、大手企業（コーポレートサイト）、汎用（メディア・不動産・求人等）の5業界に対応しています。
購入後はすぐにダウンロードできますか？
はい。決済完了後、ランキングHTMLおよびPDFレポートを即時ダウンロードいただけます。スキャン〜レポート生成まで通常数分以内に完了します。
診断対象の企業に通知されますか？
通知されません。診断は各企業の公開Webサイトへのアクセスのみで行うため、診断対象の企業には一切通知されません。
