課題

AIに見つかる前に、
正しく伝わっているか

「東京でおすすめのホテルは？」「予算10万円の高級旅館は？」のような問いに対し、ユーザーは検索だけでなくAIの回答を参照するようになっています。

この変化の中で重要なのは、単に見つかることではなく、企業情報が正確かつ一貫して伝わることです。公開性、クロール可能性、構造化データ、記述の整合性、情報の明確性が不足すると、検索エンジンやAI、ユーザーに誤って理解される可能性があります。

これは集客だけの話ではありません。社会に対する情報の信頼性の問題です。

AIの回答（情報整備あり）
帝国ホテル東京は1890年創業の老舗高級ホテルです。客室数は931室、東京都千代田区に位置し、チェックインは15:00からです。レストランは17店舗を有し、フレンチ・日本料理など多彩です。
AIの回答（情報整備不足）
〇〇旅館については詳細な情報が確認できませんでした。公式サイトをご確認ください。
仕組み

3ステップで
業界全体をスキャン

01

業界・競合を選択

ホテル・レストラン・ECなど業界を選択。知名度・高級・リスク順など抽出基準も指定できます。

02

AIスキャンを実行

各企業のWebサイトを自動取得・解析。クロール可能性、基本メタ情報、構造化データ、記述整合性を4軸で採点し、情報品質と可視性に関わる状態を整理します。

03

レポートを生成・活用

企業別PDFレポートと比較用データを生成。改善提案や現状把握の資料として活用できます。必要に応じてランキングHTMLも生成できます。

評価基準

4軸・100点満点の
スコアリング

0 — 30点

クロール可能性

HTTP 応答、robots / noindex、canonical、主要URLの取得可否など、検索エンジンやAIが情報へ到達できる前提条件を評価します。

0 — 25点

基本メタ情報

title、meta description、OGP など、ページの要約と共有に関わる基本情報を評価します。検索結果やAI要約の基礎になる項目です。

0 — 25点

構造化データ

JSON-LD や schema.org の実装状況を評価します。スキーマ型の適合性とフィールド充実度は、この軸の内訳として扱います。掲載や引用を保証するものではありません。

0 — 20点

記述整合性

企業名、業種、住所、営業時間などの主要情報がページ内で一貫しているかを確認します。人にもAIにも誤読されにくい状態かを見る軸です。

構造化データの内訳として、スキーマ型の適合性とフィールド充実度を確認します。独立した評価軸ではありません。

方法論

指数ではなく、方法論まで公開する

AI Visibility Index は、公開Webサイトを対象に、クロール可能性、基本メタ情報、構造化データ、記述整合性を4軸で評価します。業界平均や改善優先順位を示す場合も、対象範囲、評価項目、除外条件を明示します。

現行配点は、クロール可能性 30 点、基本メタ情報 25 点、構造化データ 25 点、記述整合性 20 点です。スキーマ型の適合性とフィールド充実度は、構造化データ軸の内訳として扱います。

よくある質問

FAQ

スコアはどのように計算されますか？
各企業の公式Webサイトを自動取得し、クロール可能性、基本メタ情報、構造化データ、記述整合性の4軸を解析して100点満点で採点します。検索エンジンやAIが企業情報を理解しやすい状態かを評価する指標です。詳細は方法論ページで公開しています。
診断はいつのデータを見ていますか？
リアルタイム監視ではありません。スキャン実行時点の公開Web情報をオンデマンドで取得・解析する、最新スナップショット評価です。サイトが更新されれば次回スキャン時に反映されます。
どの業界に対応していますか？
現在、ホテル・旅館、レストラン・飲食、ECサイト、大手企業（コーポレートサイト）、汎用（メディア・不動産・求人等）の5業界に対応しています。
Starter β はすぐに使い始められますか？
現在は先行案内受付中です。受付内容と対象条件を確認のうえ、通常24時間以内を目安に初回連絡を行います。正式な即時申込フローは準備中です。
診断対象の企業に通知されますか？
通知されません。診断は各企業の公開Webサイトへのアクセスのみで行うため、診断対象の企業には一切通知されません。
SemrushのAI Visibility Indexとは違うのですか？
違います。Semrushの同名指標はAI上の言及や露出の観測に近く、当サービスは公開Webサイトの情報品質診断です。AI上でどれだけ見られたかではなく、AIや検索エンジンに誤解されにくい情報整備状態かを診ます。
AIでの掲載や集客成果は保証されますか？
保証しません。本サービスは公開Web上の情報整備状況と改善余地を診断するものであり、特定のAIサービスでの掲載、引用、順位、問い合わせ増加、売上増加等を保証するものではありません。
